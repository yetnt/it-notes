# Other types of caching

## Disk Cache

A hardware mechanism for improving the time it takes to read
from or write to a hard disk.

## Browser Cache

This cache stores all files you have downloaded from the web
including web pages, images, music, videos and it also stores
all cookies.

## Web Cache

Or proxy cache, stores recently accessed web pages on a server
in a network. While browser caches are private, web cache is a
shared store of previously downloaded web pages, images, etc on
a server.
